# Recovery Periods (Class Life)

## Definition

The recovery period is the number of years over which the cost of an asset is recovered through depreciation deductions. Recovery periods are primarily determined by Revenue Procedure 87-56, which provides comprehensive guidance on class lives and recovery periods for various asset types.

## Common Class Lives

### 5-Year Property
- Computers and peripheral equipment
- Certain specialized equipment

### 7-Year Property
- Office furniture, fixtures, and equipment
- Most general business equipment

### 15-Year Property
Land improvements such as:
- Sidewalks
- Parking lots
- Landscaping

### 27.5-Year Property
- Residential rental property

### 39-Year Property
- Non-residential real property (commercial buildings)

## Table B-1 vs. Table B-2

Revenue Procedure 87-56 provides two key classification tables:

**Table B-1:** Asset-specific lives
- Provides class lives for specific asset types
- Examples: office furniture, computers, machinery

**Table B-2:** Industry-specific activities
- Provides class lives based on the business activity or industry
- Generally takes priority if an asset is used in a specific industry described in this table
